\# Trader Behavior vs Market Sentiment Analysis



This project analyzes how trader behavior differs under Fear and Greed market conditions using historical trading data and the Bitcoin Fear \& Greed Index.



\## Contents

\- Google Colab notebook with full analysis

\- Cleaned datasets and processed outputs

\- Final PDF report summarizing insights



\## Notebook

Colab Link: https://colab.research.google.com/drive/1RmlYkNO9olBIH2JBQdQnDzFrtvzqmon-?usp=sharing

Due to GitHub file size limitations, large processed CSV files are not included in this repository.



They can be regenerated by running the provided Google Colab notebook using the original datasets:



\- Historical Trader Data: https://drive.google.com/file/d/1IAfLZwu6rJzyWKgBToqwSmmVYU6VbjVs/view

\- Fear \& Greed Index: https://drive.google.com/file/d/1PgQC0tO8XN-wqkNyghWc\_-mnrYv\_nhSf/view





\## Key Insights

\- Traders take larger risks during Fear but achieve lower risk-adjusted returns

\- Greed periods show higher profitability and win rates

\- SELL-side trades outperform BUY-side trades across sentiments
